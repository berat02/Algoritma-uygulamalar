1’den 100’e kadar tek sayıları yazdıran java script ve algoritma'yı yapınız.
Algoritma
1) Başla
2) i = 1
3) Eğer !(i % 2 == 0)
4) i = ++1
5) Eğer i < = 100 ise
6) git adım 3 e
7) Bitir
Java Script

function tek_sayilari_yaz(){
var test="";
for(var i=0;i<=100;i++){
if(!(i % 2) == 0){
test += i+" ";
document.getElementById("sonuc").innerHTML = test;
}
}
}
