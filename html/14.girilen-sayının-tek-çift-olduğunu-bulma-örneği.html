Kullanıcıdan alınan sayının
tek ya da çift olduğunu kontrol edip ekranda yazdıran algoritmayı oluşturun
Algoritma
1) Başla
2) sayı gir gir
3) sayı %2 =0
4) sayı çift'tir
5) değilse
6) sayı tek'tir
7) Yazdır
8) Bitir
Java Script

function tek_cift_bulma(){
var sayi_element= Number(document.getElementById('sayi').value);
var sonuc = document.getElementById("sonuc");
if(sayi_element %2 == 0){
sonuc.innerHTML = ("Çift Sayıdır")
}else{
sonuc.innerHTML = "Tek Sayıdır"
}
}
