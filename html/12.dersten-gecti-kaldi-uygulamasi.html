Öğrencinin bir dersten aldığı not klavyeden girilerek
başarı durumu ekrana geçti veya kaldı şeklinde yazan uygulamanın algoritmasını oluşturunuz .
Algoritma
1) Başla
2) not gir
3) Eğer(not > 50 )
4) Öğrenci Dersten Geçti
5) değilse
6) Öğrenci Dersten Geçemedi
7) Bitir
Java Script

function geçti_hesaplama(){
var not_element= Number(document.getElementById('not').value);
var sonuc = document.getElementById("sonuc");
if(not_element >= 50){
sonuc.innerHTML = "Tebrikler Dersten Geçtiniz";
sonuc.style.color="green";
}else{
sonuc.innerHTML = "Üzgünüm Dersten Geçemediniz";
sonuc.style.color="green";
}
}
