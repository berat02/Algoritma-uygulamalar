Bir ürünü alış fiyatı üzerinden klavyeden
vergi oranı ve kar oranı eklenerek satış fiyatına hesaplayan programın algoritması
Algoritma
1) Başla
2) satış fiyatı giriniz
3) vergi giriniz
4) kar giriniz
5) Satış fiyat = fiyat + (fiyat * vergi/100)+(fiyat * kar/100)
6) yazdır
7) Bitir
Java Script

function hesapla(){
var toplam = Number(document.getElementById('toplam').value);
var ortalama = Number(document.getElementById('ortalama').value);
var toplama = 0;
var ortalama = 0;
for(var i = 1; i <= 5 ; i++){
toplama += i;
ortalama +=i /5;
}
document.getElementById("toplam").innerHTML = toplama;
document.getElementById("ortalama").innerHTML = ortalama;
}
