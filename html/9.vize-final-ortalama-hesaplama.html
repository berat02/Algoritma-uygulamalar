Bir öğrencinin sınavda almış olduğu notlar neticesinde ders ortalaması bulan programın JavaScript ve algoritmasını yapınız.
Algoritma
1) Başla
2) vize notu giriniz
3) final notu giriniz
4) Ortalama = ( vizenotu*30/100)+(finalnotu*70/100)
5) yazdır(ortalama) 6) Bitir
Java Script

function vize_final_ortalama_hesapla(){
var vize = Number(document.getElementById('vize').value);
var final = Number(document.getElementById('final').value);
var ortalama = ( vize * 30/100) + (final * 70/100);
document.getElementById("ortalama").innerHTML = ortalama;
}
