Klavyeden girilen fiyatı, KDV(%18) ekleyerek ekrana yazdırın.
Algoritma
1) Başla
2) Fiyat giriniz(fiyat)
3) Sonuc = fiyat * 1.18
4) Yaz sonuc
5) Bitir
Java Script

function fiyata_kdv_ekle(){
var fiyat = Number(document.getElementById('fiyat').value);
var sonuc = document.getElementById('sonuc');
var kdv = fiyat * 1.18;
sonuc.value = (kdv);
}
